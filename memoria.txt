Paso 1
~~~~~~
Para hacer las particiones hay que usar la herramienta fdisk, con permisos de superusuario.
Para crear una nueva se selecciona la opcion n. Las hemos hecho primarias las dos. El numero de
bloque de inicio y final lo hemos elegido para que quedase el disco dividido en dos aproximadamente.
Esta es la salida de pantalla con el comando # sfdisk -l.
   Disp.  Inic. Princ.   Fin   Nºcil    Nºbloq.   Id  Sistema
/dev/sdb1          0+     68-     69-    547552   83  Linux
/dev/sdb2         68+    130-     63-    500000   83  Linux

Para montar las dos nuevas particiones usamos los siguientes comandos, tras crear los directorios necesarios:
# mount -t ext3 /dev/sdb1 /sdb1
# mount -t ext4 /dev/sdb2 /sdb2

Hacemos sudo less a /etc/mtab para comprobar que se han montado correctamente.
Las lineas que hemos añadido a /etc/fstab son las siguientes:
/dev/sdb1	/sdb1	ext3	defaults	0	2
/dev/sdb2	/sdb2	ext4	defaults	0	2
